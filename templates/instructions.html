Translation Tool Instructions

1. File Requirements

Input File Format (CSV)

  • Required columns: "Product ID", "SKU", "Description"
  • File must be in CSV format with comma (,) as separator

Examples File Format (CSV)

  • Must have "Display Name" column with original text
  • Translation columns must follow format: "Display name - [language code]"
  • Language codes are configured in the Language Configuration page

Example Translation File:

Display Name Display name - sv Display name - no Display name - da

2. Language Configuration

  • Available languages are managed in the Language Configuration page
  • Each language must have a unique language code (e.g., 'sv' for Swedish)
  • The language codes in your Examples File must match those in the configuration
  • Go to Language Configuration to add, remove, or modify available languages
